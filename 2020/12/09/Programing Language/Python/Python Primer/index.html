List 和 Dict 的对比

和list比较，dict有以下几个特点：

  • 查找和插入的速度极快，不会随着key的增加而变慢；
  • 需要占用大量的内存，内存浪费多。

而list相反：

  • 查找和插入的时间随着元素的增加而增加；
  • 占用空间小，浪费内存很少。

所以，dict是用空间来换取时间的一种方法。

dict可以用在需要高速查找的很多地方，在Python代码中几乎无处不在，正确使用dict非常重要，需要牢记的第一条就是dict的key必须是不可变对象。

由于函数也是一个对象，而且函数对象可以被赋值给变量，所以，通过变量也能调用该函数。
